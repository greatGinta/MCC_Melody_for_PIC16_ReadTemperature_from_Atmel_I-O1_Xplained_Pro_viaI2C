# MCC Melody for PIC16 - Read Temperature via I2C

Read temperature data from **Atmel I/O1 Xplained Pro** extension board  
using **I2C communication** with PIC16, configured via MCC Melody.

## 📹 Video Tutorial
https://www.youtube.com/watch?v=gFbSqNg3Mng

## 🛠️ Tools & Hardware
- MPLAB X IDE
- MCC Melody
- PIC16F18446
- PICkit
- Atmel I/O1 Xplained Pro

## 📋 What it does
- Reads temperature from the onboard sensor (AT30TSE758) via I2C
- Outputs the result to terminal via UART
